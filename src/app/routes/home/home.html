A simple odt templater
for node.js and the browser

🚀 Super lightweight: 2 KB (zipped 659 bytes)
npm i odt-templater

Example

Hello {{ "{" }} user.name {{ "}" }},


{{ "{" }} #user.age {{ "}" }}

Your age is {{ "{" }} user.age {{ "}" }}

{{ "{" }} / {{ "}" }}


{{ "{" }} #user.name == John {{ "}" }}

Hello John!

{{ "{" }} / {{ "}" }}

⚙️ Features

  • Placeholder replacement
  • Conditionals
  • Empty conditionals
